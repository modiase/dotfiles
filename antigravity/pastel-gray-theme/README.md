# Pastel Gray Theme

A soft dark theme with pastel accent colours for Antigravity/VSCode.

## Installation

### Option 1: Package as .vsix

```bash
cd ~/dotfiles/antigravity/pastel-gray-theme
pnpm dlx @vscode/vsce package
antigravity --install-extension pastel-gray-theme-1.0.0.vsix
```

### Option 2: Symlink

```bash
ln -s ~/dotfiles/antigravity/pastel-gray-theme ~/.antigravity/extensions/pastel-gray-theme
```

Then activate via Command Palette: `Preferences: Color Theme` â†’ **Pastel Gray**
